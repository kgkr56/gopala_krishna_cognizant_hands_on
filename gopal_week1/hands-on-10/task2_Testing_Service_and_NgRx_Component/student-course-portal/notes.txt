KEY TESTING CONCEPTS
--------------------
HttpClientTestingModule   — test double for HttpClient (no real HTTP)
HttpTestingController     — intercepts + responds to HTTP calls in tests
httpMock.expectOne(url)   — asserts one request was made to this URL
req.flush(body, opts?)    — simulate success or error response
httpMock.verify()         — assert no unsatisfied HTTP requests (call in afterEach)

provideMockStore(opts)    — replaces NgRx Store with controllable mock
MockStore.setState(state) — replace entire state, fires all selectors
store.resetSelectors()    — clear overrideSelector() overrides between tests
dispatchSpy               — spyOn(store, 'dispatch') to assert actions fired
