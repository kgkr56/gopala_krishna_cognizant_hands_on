  Install "Redux DevTools" from Chrome/Firefox extension store.

WHAT WAS BUILT
--------------

  — Step 95: provideState('course', courseReducer) registers the feature slice
  — Step 92: provideStoreDevtools({ maxAge: 25 }) enables Redux DevTools

pages/course-list/
  — Step 96: dispatches loadCourses() on ngOnInit
  — selects courses$, loading$, error$ via store.select()
  — renders via async pipe (no manual subscribe/unsubscribe)

KEY CONCEPTS
------------
[Course] prefix convention   — groups actions by feature in DevTools timeline
Selectors + memoisation      — only recompute when input slice changes
async pipe                   — subscribes, unsubscribes, and triggers CD automatically
Pure reducer                 — no side effects; only action + state → next state
